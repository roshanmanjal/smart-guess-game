# Jarvis Assistant (Python)

## Overview
A voice-controlled personal assistant that performs basic system and web tasks.

## Features
- Open applications (Chrome, Notepad)
- Voice commands using speech recognition
- Text-to-speech responses
- Google search integration
- Time and date information

## Tech Stack
- Python
- speech_recognition
- pyttsx3
- webbrowser
- os

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the assistant:
   python main.py
